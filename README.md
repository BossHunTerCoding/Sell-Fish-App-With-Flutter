# Sell Fish Flutter

โปรแกรมสำหรับคำนวนราคาปลาตามน้ำหนัก (กิโลกรัม)

# คู่มือการคำนวน

1 กิโลกรัม เท่ากับ 10 ขีด
1 ขีด เท่ากับ 100 กรัม
1 กิโลกรัม เท่ากับ 1,000 กรัม
1 เมตริกตัน เท่ากับ 1,000 กิโลกรัม

# วิธีการคำนวนราคา

นำราคาปลาคูณกับน่ำหนักของปลาที่ได้ (กิโลกรัม)
